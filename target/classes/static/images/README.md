# Imágenes para el Combat Simulator

Coloca aquí tus imágenes personalizadas:

| Archivo | Descripción | Tamaño recomendado |
|---------|-------------|---------------------|
| `background.jpg` o `background.png` | Fondo de la escena de batalla | 1920×1080 o similar |
| `player.png` o `player.jpg` | Sprite/imagen del héroe | 128×128 o 256×256 |
| `enemy.png` o `enemy.jpg` | Sprite/imagen del enemigo (dragón) | 128×128 o 256×256 |

Si no añades imágenes, se usarán emojis y colores por defecto.
